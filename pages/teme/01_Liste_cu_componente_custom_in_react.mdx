## Realizarea unei liste de componente custom in React.js

### Cerinte:
- Initializeaza un nou Codespace React pe GitHub
- Sterge html-ul exemplu `App.jsx`
- Sterge continutul exemplu din `App.css`
- Creaza un container pentru elementele listei
- Scrie de mana niste elemente ale listei folosind eticheta `li`
- Transforma elementele listei in componente custom separate de functia principala `App`
- Inlocuieste elementele listei cu mai multe componente custom
- Intr-o variabila lista numita `elemente`, minim 3 valori de tip textului
- Folosind functia `elemente.map((element_curent)=>{})` inlocuieste componentele din lista din html cu elementele din variabila creata anterior.

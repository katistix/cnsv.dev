# Formatarea Textului și Elementele de Bază în HTML

Această lecție explorează cum să folosim tag-urile HTML pentru formatarea textului și pentru a introduce elemente de bază, precum liste și tabele, în paginile web.

## Formatarea Textului

### 1. Titlurile

HTML oferă șase niveluri de titluri, de la `<h1>` la `<h6>`. `<h1>` este cel mai important și este folosit pentru titlul principal al paginii.

```html
<h1>Titlu Principal</h1>
<h2>Subtitlu</h2>
...
<h6>Titlu Nivel Mic</h6>
```

### 2. Paragrafe

Elementul `<p>` este folosit pentru a crea un paragraf.

```html
<p>Acesta este un paragraf.</p>
```

### 3. Textul Îngroșat și Italic

Pentru a îngroșa textul, folosiți `<strong>`, iar pentru italic `<em>`.

```html
<strong>Text Îngroșat</strong>
<em>Text Italic</em>
```

### 4. Citate

Pentru citate, se folosește `<blockquote>`, care de obicei este reprezentat într-un stil diferit de textul obișnuit.

```html
<blockquote>Acesta este un citat.</blockquote>
```

## Liste

### 1. Liste Neordonate

Listele neordonate folosesc elementul `<ul>` cu elemente de listă `<li>`.

```html
<ul>
    <li>Element 1</li>
    <li>Element 2</li>
    <li>Element 3</li>
</ul>
```

### 2. Liste Ordonate

Listele ordonate folosesc `<ol>` și funcționează similar cu `<ul>`, dar afișează elementele într-o ordine numerică.

```html
<ol>
    <li>Primul Element</li>
    <li>Al Doilea Element</li>
</ol>
```

## Tabele

Tabelele sunt create cu `<table>`, iar rândurile cu `<tr>`. Fiecare celulă este definită de `<td>` (celulă normală) sau `<th>` (celulă de antet).

```html
<table>
    <tr>
        <th>Antet 1</th>
        <th>Antet 2</th>
    </tr>
    <tr>
        <td>Celulă 1</td>
        <td>Celulă 2</td>
    </tr>
</table>
```

## Concluzie

Aceste elemente de bază oferă uneltele necesare pentru a începe structurarea și formatarea conținutului în paginile HTML. Prin experimentarea și practica acestor elemente, veți putea crea pagini web mai structurate și vizual plăcute.
